TF 坐标变换（跟踪小海龟例程）
发表于 2019-12-02| 更新于 2020-01-08|学习笔记
字数总计:790|阅读时长: 3 分钟
| 阅读量:

笔记整理记录自：https://www.bilibili.com/video/av69417218

1、新建工作空间

  1. 新建 catkin_ws 文件：mkdir -p catkin_ws/src
  2. 进入 catkin_ws 目录：cd catkin_ws
  3. 初始化：catkin_make
  4. 进入 src 文件：cd src
  5. 创建用于通信的 package：catkin_create_pkg learning_tf std_msgs rospy roscpp
  6. 编译一下：catkin_make
  7. 添加 source：echo "source ~/catkin_ws/devel/setup.zsh " >> ~/.zshrc
  8. 返回根目录：cd
  9. 查看 source 是否添加：gedit .zshrc 或者 vi .zshrc

2、创建编程文件

  1. 找到刚才的工作空间：cd ~/catkin_ws/src/learning_tf/src
  2. 新建 turtle_tf_broadcaster.cpp 文件：gedit turtle_tf_broadcaster.cpp
  3. 保存如下代码：
c++

  1. 新建 turtle_tf_listener.cpp 文件：gedit turtle_tf_listener.cpp

  2. 保存如下代码：

c++

3、配置

  1. 找到刚才的工作空间：cd ~/catkin_ws/src/learning_tf/
  2. 编辑 CMakeLists.txt 文件：gedit CMakeLists.txt
  3. 保存如下代码：
Code
  1. 返回主目录：cd ~/catkin_ws/
  2. 编译一下：catkin_make

4、创建 launch 文件

  1. 找到刚才的工作空间：cd ~/catkin_ws/src/learning_tf/
  2. 新建 launch 文件夹：mkdir launch
  3. 新建文件：gedit start_demo_with_listener.launch
  4. 保存如下代码：
html

5、运行

  1. 启动 launch 坐标文件：roslaunch learning_tf start_demo_with_listener.launch
  2. 通过按键控制海龟即可
