ROS Melodic 小海龟操控实例

2019-12-02 学习笔记 ROS 0 评论 字数统计: 281(字) 阅读时长: 1(分)

执行新的命令操作时，均需要启动一个新的终端然后输入命令

基础操作：

  1. 启动 ROS Master：roscore
  2. 启动小海龟仿真器：rosrun turtlesim turtlesim_node
  3. 启动小海龟键盘控制节点：rosrun turtlesim turtle_teleop_key
  4. 此时按键盘上的方向键即可操控小海龟运行

其他操作：

  1. 新生一只小海龟：rosservice call /spawn "x:5.0 y:5.0 theta:0.0 name:'turtle2'"
  2. 按指定命令控制小海龟：rostopic pub / turtle/cmd_vel geometry_msgs/Twist ...
    1. 代码中的省略号里面包括了坐标及角度等信息，自行修改即可
    2. 以一定的频率控制（1s 10次）：rostopic pub -r 10 / turtle/cmd_vel geometry_msgs/Twist ...
    3. 控制另外一只小海龟：ostopic pub -r 10 / turtle2/cmd_vel geometry_msgs/Twist ...

一只小海龟追赶另外一只小海龟实例：

  1. 安装小海龟坐标变换功能包：sudo apt install ros-melodic-turtle-tf
  2. 运行功能包：roslaunch turtle_tf turtle_tf_demo.launch
  3. 启动小海龟键盘控制节点：rosrun turtlesim turtle_teleop_key
  4. 此时按键盘上的方向键即可操控第一只小海龟运行，第二只小海龟会自动跟踪第一只小海龟
